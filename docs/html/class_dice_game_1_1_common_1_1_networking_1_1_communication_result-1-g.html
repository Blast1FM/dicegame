Detailed Description

Implementation of a communication result, used in ReceiveMessage Represents 2 outcomes - a succesful message exchange or an error If successful - Result is set to DataExchangeResult.Ok and the message is set, while the ErrorMessage remains null If an error occurs - Result is set to DataExchangeResult.Error and the ErrorMessage is set, while the Message property is null. The packet property is set regardless, to allow inspection of the request packet.
